# Sistem-Laundry-Kel-5-FPW

Web-based laundry system project for FPW final exam. Features include online ordering, real-time tracking, QRIS/VA statics, driver pickup, daily reports, and complaint handling. Built with Laravel by Informatics Group 5, Universitas Singaperbangsa Karawang
